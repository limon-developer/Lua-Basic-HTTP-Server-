# Lua-Basic-HTTP-Server-
Lua ile Basit HTTP Server projesine hoşgeldin bu proje HTTP sunucuları açmak için yapılan bir repo' dur

başlangıç
```bash
luarocks install LuaSocket
```
İP ile Çalıştırmak için

```IP İle Başlat
lua ip-server.lua
```
LAN'da Çalıştırmak için 
```LAN İle Başlat
lua LAN.lua
```
127.0.0.1(localhost) ile Çalıştırmak için
```localhost İle Başlat
lua localhost.lua
```
